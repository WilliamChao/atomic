ビルドに必要な手順を以下にまとめます。


・CUDA Toolkit と GPU Computing SDK をインストール
- http://developer.nvidia.com/cuda-toolkit-40
- CUDA 4.0 (cudatoolkit_4.0.17_win_64.msi と gpucomputingsdk_4.0.19_win_64.exe) を想定


・cygwin
- カスタムビルドステップで cygwin 上の ruby を使っているため、cygwin のインストールが必要です。
- http://cygwin.com/
- 環境変数 CYGWIN_ROOT にインストールディレクトリを指定 (例: C:\cygwin)

* 環境変数の変更はログアウトか再起動するまで反映されないかもしれません
