・CUDA Toolkit と GPU Computing SDK をインストール
- http://developer.nvidia.com/cuda-toolkit-40
- CUDA 4.0 (cudatoolkit_4.0.17_win_64.msi と gpucomputingsdk_4.0.19_win_64.exe) を想定

・boost
- インストーラを利用した方が楽 http://www.boostpro.com/download/ BoostPro 1.47.0 Installer を想定
- 環境変数 BOOST にインストールパスを設定 (例: C:\Program Files (x86)\boost\boost_1_47)

・cygwin
- ruby をインストール http://cygwin.com/
- 環境変数 CYGWIN_ROOT にインストールディレクトリを指定 (例: C:\cygwin)

* 環境変数の変更はログアウトか再起動するまで反映されない
以上でビルドできるようになるはず。
